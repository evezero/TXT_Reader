# TXT Reader v1.2

精致、快速的 Windows 原生级 TXT 电子书阅读器。基于 Tauri 和 React 开发，为你带来纯粹而专注的阅读体验。

## 🌟 核心特性 (v1.2 全新升级)

- **一比一原生渲染**：彻底重构了文本分段与行号引擎，完全保留原文档的所有空行。实现与 Windows 原生记事本 100% 对应的精准行号映射。
- **智能目录防漏判**：改进了目录提取算法，大幅降低去重阈值，确保类似“十四”这种前后只有两三句对白的超短章节也能被稳妥收录进目录。
- **精准高亮与平滑跳转**：重新设计了滚动侦测核心算法（`bottom > scrollTop`），修复了平滑滚动造成的焦点错位问题，现在跨度再大的目录跳转也能一次性精准高亮，指哪打哪。
- **安全的主动保存机制**：移除了极其危险的光标移开自动保存逻辑，加入了**未保存状态提示（`*` 号）**。你可以通过顶部的保存按钮或快捷键 `Ctrl+S` 手动保存；如果试图关闭未保存的标签页，原生安全弹窗会及时拦截并询问。
- **轻便的交互体验**：新增了常驻的关于/版本信息面板；原有的加号按钮升级为智能跳转（空页面选文件，阅读状态回书架）。

## 🚀 基础特性

- **阅读设置**：字号、行距、背景色全方位可调。
- **双页对比**：支持分屏对照阅读不同文件。
- **摸鱼模式**：提供极致隐蔽的无边框、半透明阅读挂件。
- **自动进度记忆**：自动记录你上一次的阅读位置和所有书签。

## 📦 如何编译

本项目依赖于 [Node.js](https://nodejs.org) 和 [Rust](https://rustup.rs/)。

1. 安装依赖：
   ```bash
   npm install
   ```

2. 开发环境运行：
   ```bash
   npm run dev
   ```

3. 编译发布为单文件 EXE（在 Windows 上需要 Visual Studio C++ Build Tools）：
   ```bash
   npm run tauri build
   ```
   编译成功后，可执行文件位于 `src-tauri/target/release/txt-reader.exe`。
