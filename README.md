# TXT Reader

TXT Reader 是一款轻量级长文本阅读与编辑工具，基于 Tauri + React 架构开发，致力于提供纯净的阅读体验与类 Notepad（记事本）的无缝编辑能力。

## 核心功能

### 1. 深度集成的编辑引擎
突破传统阅读器“只读”限制，提供与 Windows 记事本一致的文本编辑体验：
- **原生级段落编辑**：支持使用 `Enter` 换行拆分、`Backspace` 和 `Delete` 跨段落合并，实现平滑的光标交互。
- **完善的修改保护**：实时脏状态（Dirty State）监控与未保存提示，退出或切换文件时防止修改丢失。

### 2. 独立阅读进度管理
- **自动进度记忆**：阅读进度独立于文本文件存储，自动记忆最后阅读的章节与精确到行的滚动位置。

### 3. 高效目录提取与清洗
- **智能正则解析**：内置正则匹配引擎，快速从海量文本中提取符合特征的章节标题。
- **动态去重与清理**：支持右键手动剔除识别错误的冗余目录项，系统将记忆用户的去重选择。

### 4. 生产力阅读模式
- **双栏分屏模式**：支持一键文档分屏，便于前后文对照、左右精读。
- **透明摸鱼模式**：提供高隐蔽性的纯透明无边框窗口模式，便于融入各类工作桌面环境。
- **高度定制化排版**：支持多款护眼主题，可自由调整行距、字体及段落缩进。

## 部署与编译

本项目基于 `Tauri + React + Vite` 构建：

```bash
# 1. 安装项目依赖
npm install

# 2. 编译 Windows 可执行文件 (.exe)
npm run tauri build
```
构建的单文件程序位于 `src-tauri/target/release/txt-reader.exe`。
