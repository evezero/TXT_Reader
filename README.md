# TXT Reader

精致、快速的 Windows 原生级 TXT 电子书阅读器。基于 Tauri 和 React 开发，为你带来纯粹而专注的阅读体验。

## 🌟 核心特性

- **极致的大文件性能**：全量秒加载数十兆级的 TXT 文件，提供如原生般顺滑的全局滚动体验，告别虚拟列表带来的光标卡顿问题。
- **一比一原生渲染**：完美还原源文件的结构排版，精准保留所有空行与分段，实现与 Windows 记事本完全一致的段落及行号映射。
- **智能目录提取**：内置强大的正则解析引擎，能够精准识别中英文各类章节标题（如“第一章”、“Chapter 1”甚至无前缀的短数字章节）。自带智能去重与防冗长误判机制，确保生成的目录纯净可靠。
- **平滑且精准的跳转**：不论目录跨度多大，点击章节均可实现平滑滚动，并精准计算滚动边界，高亮一次性完美贴合当前阅读进度。
- **安全的主动保存**：提供可视化的保存按钮与快捷键 (`Ctrl+S`) 机制，并在标签页名称上显示 `*` 号修改提示。当试图关闭未保存的文件时，原生安全弹窗会拦截并询问，彻底杜绝数据意外丢失或覆盖风险。
- **更人性化的工作流**：支持多标签页管理；内置“书架”系统方便快速切换历史文档；智能响应工具栏点击，提供流畅的文件流转体验。

## 🚀 基础功能

- **深度自定义阅读界面**：字号、行距、字体及段落间距全方位可调。
- **双页同屏对比**：支持横向分屏，轻松对照阅读多个不同文件。
- **多种主题切换**：内置标准、护眼黄底、夜间暗灰等主题，完美适配系统深色模式。
- **隐蔽的摸鱼模式**：提供极致隐蔽的无边框、半透明悬浮挂件阅读模式。
- **自动进度记忆**：退出后自动保存你的阅读位置及全部自定义书签，下回打开无缝衔接。

## 📦 如何编译

本项目依赖于 [Node.js](https://nodejs.org) 和 [Rust](https://rustup.rs/) 开发环境。

1. 安装依赖：
   ```bash
   npm install
   ```

2. 开发环境运行：
   ```bash
   npm run dev
   ```

3. 编译为单文件 EXE（在 Windows 上需要 Visual Studio C++ Build Tools）：
   ```bash
   npm run tauri build
   ```
   编译成功后，可执行文件位于 `src-tauri/target/release/txt-reader.exe`。
